# 🌌 Aurora Drift Interface — Portfolio of Amrita Malik

🚀 A next-generation interactive portfolio experience designed as a **living digital environment**, where UI elements behave like they exist in a soft gravitational field of motion, light, and energy.

---

## ✨ Overview

This project is a futuristic personal portfolio built to go beyond traditional websites.

Instead of static sections, it creates a **dynamic experience system** where every element floats, reacts, and evolves inside a layered digital space.

The interface is inspired by immersive motion design, sci-fi aesthetics, and interactive 3D environments.

---

## 🧠 Concept

> “A digital identity system where design, motion, and interaction exist in a continuous flowing environment.”

This portfolio focuses on:
- Depth over flat design
- Motion over static layout
- Experience over pages
- Identity over templates

---

## 🌌 Features

- 🧲 Magnetic Drift UI interaction system  
- 🌫️ Aurora-style animated background (layered energy waves)  
- 🐈 3D cyber cat hero element (signature identity anchor)  
- 💜 Premium dark + purple futuristic aesthetic  
- 🎭 Smooth scroll-based transitions  
- 🧠 Interactive project showcase system  
- 📱 Fully responsive design (mobile + desktop optimized)
